Project Title : Finding Doctors - Hackthon Project

Start Date : 16-02-2024

End Date   : 29-02-2024

Project Description 

1. Navigate to Practo Homepage.
2. Finding Doctors and apply some filters.
3. Display Top 5 Doctors List.
4. Display all Top Surgeries List.
5. Navigate to the Health Wellness form.
6. Fill the required data.
7. Verify Form Successfully Submitted.
8. If Entered Invalid data Submit Button Disabled or Not. 

Step to excute 

TestNG :-
-----------------------------------------------------

Testing   : click master.xml file then click to run the test.

Regression Test  : click Regression.xml file then click to run the test.

Smoke Test : click smoke.xml file then click to run the test.



Cucumber Test :-
-----------------------------------------------------

Testing   : click TestRunner.java file then click to run the test.

Regression Test  : click Regression_TestRunner.java file then click to run the test.

Smoke Test : click smokeTest_TestRunner.java file then click to run the test.
